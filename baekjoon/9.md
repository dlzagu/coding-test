### Q.2739
-----
>N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오. 출력 형식에 맞춰서 출력하면 된다.

* 코드 
```py
a = int(input())

for i in range(1,10):
    print(a, '*', i, '=', (a*i))
```

### Q.10950
-----
>문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오. 

>입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다.
각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

*코드 
```py
a = int(input())

for i in range(a):
    a,b = map(int, input().split())
    print(a+b)
```
